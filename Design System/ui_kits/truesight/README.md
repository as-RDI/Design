# TrueSight UI Kit — RDI Technologies

Recreation of the TrueSight analysis product UI (in-app chrome).

Components:
- `AppChrome.jsx` — full app shell: topbar, left rail, right inspector, footer status
- `VideoCanvas.jsx` — the MA video with overlaid measurement points + amplified-motion vectors
- `AssetList.jsx` — left rail list of detected assets (TrueSight ID)
- `SpectrumPanel.jsx` — bottom frequency-domain spectrum
- `InspectorPanel.jsx` — right-side properties/severity for the selected point
- `Toolbar.jsx` — top toolbar with play/record/amplify controls
